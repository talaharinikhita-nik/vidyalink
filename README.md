# VidyaLink 📚
India's privacy-first AI-powered tutoring platform.

🔗 Live Demo: https://vidyalink-app.streamlit.app

## What it does
- Students type in plain English to find tutors — AI matches them by subject, location, and budget
- Role-based dashboards for students and tutors
- Privacy-first — no Aadhaar, no PAN required
- Real cloud database with Supabase

## Built with
- Python, Streamlit
- Groq LLM (Llama 3)
- Supabase (PostgreSQL)
- Deployed on Streamlit Cloud
